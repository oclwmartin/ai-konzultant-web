# AI Konzultant Landing Page

Produkčně připravená landing page pro AI automation konzultanta.

## Stack
- Single static HTML soubor
- Vanilla CSS (vlastní design systém, dark/light mód)
- Vanilla JS (jazykový přepínač, animace, dark mód)
- Žádné build nástroje, žádné závislosti

## Nasazení
Otevři `index.html` přímo v prohlížeči NEBO nasaď na libovolný static host (Netlify, GitHub Pages, Vercel).

## OpenClaw integrace
Toto repo monitoruje OpenClaw agent pro aktualizace obsahu a deployment triggery.
